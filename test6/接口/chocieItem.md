# 接口：chocieItem
  [点击返回](../README.md)
用例： [](../用例/登录用例.md)

- 接口名称：choiceItem

- 功能：
  选择学期查看成绩。

- API请求地址：
  /v1/api/chocieItem

- 请求方式 ：
  GET

- 请求参数说明:
  以ID为查找参数，查找不同学生的成绩

- 返回实例：

      {
          "status": true,
          "Item"：[
              "2015-2016第一学期",
          ]
      }

- 返回参数说明：

|参数名称|说明|
|:---------:|:--------------------------------------------------------|
|status|布尔类型，true表示正确的返回，false表示有错误|
|Item|学期名称|
