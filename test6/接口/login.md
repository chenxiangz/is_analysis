接口：login
--
[点击返回](../README.md)      \
用例： [登录](../用例/登录用例.md)
- 功能：
    登录平台。

- 权限：
    所有由登陆账号的用户。

- API请求地址：
    接口基本地址/v1/api/login

- 请求方式 ：
    POST

- 请求实例：

        {
            "id":"201510414104",
            "password":"sixsixsix",
            "type":"学生"
        }

- 请求参数说明:

  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|
  |id|用户的id|
  |password|用户的密码|
  |type|用户类型，学生或者老师|

- 返回实例：

        {
            "status": true,
            "infor": null,
        }

- 返回参数说明：

  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|
  |status|布尔类型，true表示正确的返回，false表示有错误|
  |infor|返回结果说明信息|
