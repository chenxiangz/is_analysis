接口：addStuCourse
---------------
 [点击返回](../README.md)   \
用例： [添加课程信息](../用例/添加课程信息.md)
- 功能：
    添加学生的课程各种信息。

- 权限：
    只有使用管理员权限时，才能进行添加等操作。

- API请求地址：
    接口基本地址/v1/api/addStuCourse

- 请求方式 ：
    POST

- 请求实例：

        {
            "course_id":"01",
            "course_name":"信息系统",
        }

- 请求参数说明:

  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|
  |course_id|课程编号|
  |course_id|课程名字|

- 返回实例：

        {
            "status": true,
            "infor": null,
        }

- 返回参数说明：

  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|
  |status|布尔类型，true表示正确的返回，false表示有错误|
  |info|返回添加结果信息|
