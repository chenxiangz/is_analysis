# 接口：choiceCourse
[点击返回](../README.md) \
用例： [选课](../用例/选课.md)
- 功能：
    选择课程内容。

- 用户：
    学生。
- API请求地址：
    接口基本地址/v1/api/choiceCourse
- 请求方式 ：
    POST

- 请求实例：

        {
            "name":"政治1",
            "desc":"政治1是。。。。",
        }

- 请求参数说明:

  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|
  |name|课程名称|
  |desc|课程内容描述|

- 返回实例：

        {
            "status": true,
            "infor": null,
        }

- 返回参数说明：

  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|
  |status|布尔类型，true表示正确的返回，false表示有错误|
  |infor|返回选择结果信息|
